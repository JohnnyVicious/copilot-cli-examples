# Results Directory

Store solved challenge outputs here, separated by language at the first level:

- `results/python/` — Python 3.11+ solutions and artifacts
- `results/go/` — Go solutions and artifacts
- `results/rust/` — Rust solutions and artifacts
- `results/powershell/` — PowerShell Core solutions and artifacts

Place each challenge’s final code, notes, or run outputs in the appropriate language folder. Keep generated binaries out of version control.***
